# AIS_Policlinika
Название проекта: АИС "Электронная больница".


Цель: разработка веб-приложения, состоящее из информационной системы и системы электронной очереди для записи к врачам и лабораториям.

Суть проблемы: одна из целей данного проекта является хранение медицинских карт пациентов. Хранение медицинских карт в архивах в бумажном виде имеет ряд недостатков, и почти не имеет никаких преимуществ. В данные недостатки можно включить ненадежность бумаги, и занимаемое место. Электронная медицинская карта позволит не только хранить историю болезни и результаты анализов в электронном виде, но и позволит получить доступ к мед. карте врачам из различных больниц, которые зарегистрированы в данной системе. 

Вторая цель заключается в разработке электронной очереди к врачу. 
Живая очередь отходит на второй план, уступая своё место электронной. Она позволяет в удобной форме записываться к выбранному врачу, выбрать свободное время и дату приёма, описать симптомы,
 и подтвердить запись.
 Каждый врач имеет график работы, от которого и зависят временные варианты записи. Также, для гибкости программы, можно реализовать время чередования пациентов 
из живой и электронной очереди. 
Каждый врач выделяет пациентам из электронной очереди время приёма.

Третья цель заключается в хранении информации (прайс-лист, новости, отзывы) о больницах, 
лабораториях, аптеках, врачах, и системе в целом.


Система не привязана к конкретному медицинскому учреждению. Все учреждения, желающие сотрудничества с системой, могут быть
 зарегистрированы главным администратором. Он добавляет учреждение, и администратора для него. 
Тот, в свою очередь, добавляет модераторов и врачей.

Технологии разработки: C#, ASP.NET MVC, ADO.NET Entity Framework, MS SQL Server, HTML5, CSS3, jQuery, Twitter Bootstrap .Остальные библиотеки будут подключаться по мере 
необходимости.
